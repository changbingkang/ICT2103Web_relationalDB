How to run?

1) unzipped the project folder in ur c drive
2) go to path C:\ICT2103Web open cmd in this path
3) input this line in cmd: ICT2103Web\Scripts\activate
4) the above step is to activate the virtual env before opening the project solution. 
Activate successfully you will see (ICT2103Web) C:\ICT2103Web>
5) open visual studio code and open the project folder [C:\ICT2103Web]
6) click Terminal > new terminal
7) make sure you see this in the terminal (ICT2103Web) C:\ICT2103Web
8) to run the application
9) key in python application.py
10) pressed ctrl and click on the http link
You will see this in terminal command
